### 实施步骤

**1. 初始化 React 项目**
- 在 `web/` 文件夹创建 Vite + React 18 + TypeScript 项目
- 安装 `react-router-dom`

**2. 迁移样式系统**
- 复制现有 CSS 文件到 `web/src/styles/`
- 保持 CSS 变量和设计系统不变

**3. 创建核心组件**
- `Sidebar` - 侧边栏导航（统一管理，消除重复）
- `Footer` - 页脚
- `Layout` - 布局容器
- UI 组件：CodeBlock、Card、Tag、TipBox 等

**4. 创建页面组件（36个）**
- 将每个 HTML 页面的内容转换为对应的 React 组件
- 保持文本、数据、展示顺序完全一致

**5. 配置路由**
- 使用 React Router 配置所有页面路由
