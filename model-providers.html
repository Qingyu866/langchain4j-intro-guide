模型提供商

LangChain4j支持20+ LLM提供商和30+向量数据库，统一的API让模型切换变得轻而易举。

1

OpenAI

最流行的LLM提供商，提供GPT系列模型。

Java 
import dev.langchain4j.model.openai.OpenAiChatModel;

OpenAiChatModel model = OpenAiChatModel.builder()
    .apiKey(System.getenv("OPENAI_API_KEY"))
    .modelName("gpt-4")
    ..build();
2

Anthropic

提供强大的Claude系列模型，擅长推理和代码生成。

Java 
import dev.langchain4j.model.anthropic.AnthropicChatModel;

AnthropicChatModel model = AnthropicChatModel.builder()
    .apiKey(System.getenv("ANTHROPIC_API_KEY"))
    .modelName("claude-sonnet-4-20240520")
    .build();
3

Azure OpenAI

微软提供的云服务，企业级可靠性和安全性。

4

其他提供商

包括Hugging Face、Google Vertex AI、Amazon Bedrock等

Hugging Face

开源模型平台，支持多种开源模型

Google Vertex AI

Google的AI服务，集成多个模型

统一API优势

所有提供商使用相同的ChatModel接口，切换模型只需修改一行代码。

快速切换

从OpenAI切换到Anthropic，只需修改类名和API Key

代码复用

所有业务逻辑代码无需修改，大幅降低迁移成本
