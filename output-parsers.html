输出解析

掌握输出解析技术，将LLM的文本输出转换为类型安全的Java对象，构建数据驱动的AI应用。

1

为什么需要输出解析

LLM以文本形式返回答案，但实际应用中往往需要结构化数据，如JSON、对象、枚举等。输出解析器可以将文本转换为Java类型安全对象。

2

基本用法

LangChain4j提供了强大的输出解析功能，使用简单的注解即可实现类型转换。

Java 
import dev.langchain4j.AiServices.AiServices;
import dev.langchain4j.service.TypeSafeChatLanguageModel;
import dev.langchain4j.service.OutputParser;

interface Person {
    String name;
    int age;
}

public class OutputParser personParser = AiServices.builder(TypeSafeChatLanguageModel.builder(model))
    .outputParser(OutputParser.ofString())
    .build();

String json = model.generate("""你的名字叫什么？""");
Person person = personParser.parse(json);

代码解析： 1. 定义数据类（Person） 2. 创建AiServices.builder()并配置model和outputParser 3. 使用generate()获取文本，调用parse()转换为对象

3

支持的输出类型

LangChain4j支持多种输出解析器，覆盖常见使用场景。

文本解析

ofString() - 解析简单字符串和基本类型

Bean解析

ofBean() - 解析为Java对象（POJO）

JSON解析

fromJson() - 将JSON字符串转换为对象

List解析

fromCsv()/fromXml() - 解析列表格式数据

Enum解析

fromEnum() - 解析枚举类型

错误处理

处理解析失败和格式错误

Java 
try {
    Person person = personParser.parse(response);
    return person;
} catch (OutputParserException e) {
    System.err.println("解析失败：" + e.getMessage());
}
